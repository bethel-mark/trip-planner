# 出行必备清单模板(9 大类 · 80+ 项)

> 来源:PROJECT_MEMORY.md 第 10 节。可直接复制到任何行程规划的 checklist 中,按目的地智能剪裁。

## A. 证件类 📄
- [ ] 护照原件(有效期 ≥ 6 个月,部分国家要求 ≥ 6 个月余量)
- [ ] 签证 / 电子签(eVisa / VOA / ETA,按目的地)
- [ ] 身份证(国内段乘机 / 酒店登记备用)
- [ ] 驾照 + 国际驾照 IDP(如需租车)
- [ ] 机票 / 火车票 / 大巴 / 酒店 确认单(打印 + 电子双备份)
- [ ] 户口本翻译件(未成年人出境 / 监护公证)
- [ ] 旅行保险单(电子 + 纸质)
- [ ] 疫苗接种证明(如需黄热病 / COVID 等)
- [ ] 公司 / 学校在职 / 在读证明(商务 / 探校)

## B. 钱 & 卡 💳
- [ ] 当地货币现金(小额,机场 / 出租车 / 小店)
  - 建议:USD 200-500 等值 + 当地币零钱
- [ ] 双币 / 多币种信用卡(Visa + Master 各 1,备份)
  - **必须免外汇手续费**(华夏 / 中行全球通 / Wise)
- [ ] Travel Money Card(Wise / Revolut / 中行 EMV)
- [ ] 银联卡(部分东南亚 ATM 取当地币)
- [ ] 海外提款手续费确认(< 1.5% / 笔)
- [ ] 出境前换汇(机场 / 银行,留意汇率 + 手续费)

## C. 电子产品 🔌
- [ ] 手机 + 充电器(原厂 / MFi 认证)
- [ ] **充电宝**(飞机随身,≤ 100Wh,必须可登机)
- [ ] **转换插头**(按目的地)
- [ ] 数据线 × 2(主 + 备)
- [ ] 蓝牙耳机(飞机 / 大巴)
- [ ] 平板 / 笔记本(视行程)
- [ ] 相机 + 内存卡(可选)
- [ ] 通用读卡器
- [ ] 小型工具刀(瑞士军刀 / Leatherman)

## D. 通讯 & 网络 📱
- [ ] **漫游流量包**(国内运营商,提前 1 天开通)
  - 中移动 2 天 ¥35 / 7 天 ¥88
- [ ] 当地 eSIM(东南亚推荐 Airalo / Holafly,落地即用)
- [ ] 当地 SIM 卡(落地机场买,通常比漫游便宜)
- [ ] 重要 APP 提前装好:
  - [ ] Google Maps(海外)
  - [ ] Grab / Uber / DiDi(东南亚打车)
  - [ ] 当地打车 / 公交 APP
  - [ ] Google Translate(带离线语言包)
  - [ ] WhatsApp / Line / KakaoTalk(通讯)
  - [ ] Booking / Agoda / Airbnb(订房)
  - [ ] 航司 APP(查登机口 / 改签)
  - [ ] Wise(汇率 / 转账)
  - [ ] 当地美食 APP

## E. 健康 & 药品 💊
- [ ] **常备药**(按个人情况):
  - 感冒药 / 退烧药(对乙酰氨基酚)
  - 肠胃药(思达安 / 整肠丸 / 蒙脱石散)
  - 抗过敏药(氯雷他定)
  - 晕车药
  - 创可贴 / 消毒棉片
  - 蚊虫叮咬膏
  - 个人慢病药(降压 / 降糖 / 抗抑郁等)
- [ ] 口罩(备用)
- [ ] 消毒湿巾 / 免洗洗手液
- [ ] 防晒霜(SPF 30-50+,热带推荐 SPF 50+)
- [ ] 蚊虫驱避剂(热带 / 东南亚)
- [ ] 太阳镜
- [ ] 雨伞 / 雨衣
- [ ] 个人卫生用品(牙刷 / 毛巾 / 拖鞋,部分国家酒店不提供)
- [ ] 处方药英文翻译件(海关检查用)
- [ ] 体温计 / 血氧仪(可选)

## F. 衣物 👕
- [ ] 内衣袜子(N+1 件)
- [ ] 上衣(快干面料优先,热带 / 亚热带)
- [ ] 长裤(正式场合 / 防蚊 / 冷气)
- [ ] 短裤 / 短裙(度假)
- [ ] 1 件薄外套(冷气房 / 山区 / 飞机)
- [ ] 1 件防风防雨外套
- [ ] 1 双步行舒适的鞋(城市暴走必备)
- [ ] 1 双拖鞋 / 凉鞋
- [ ] 泳衣(度假 / 海岛)
- [ ] 帽子 / 太阳帽
- [ ] 围巾(冷气 / 清真寺等场所)
- [ ] 睡衣
- [ ] 1 套正式服装(高级餐厅 / 宗教场所)
- [ ] 按行程天数 × 1.5 计算衣量

## G. 安全 & 应急 🛡️
- [ ] 旅行保险(医疗 / 行李 / 行程取消)
- [ ] 紧急联系人清单(国内家人 + 大使馆 + 当地报警)
- [ ] 护照复印件 + 电子版(网盘备份)
- [ ] 证件照 2 张(备用)
- [ ] 行程单打印件(给家人留底)
- [ ] 现金分散存放(不同口袋 / 包)
- [ ] 防偷腰包 / 防盗背包
- [ ] 酒店地址 + 当地报警电话(中英双版)
- [ ] 海外使领馆联系方式(目的地国)
- [ ] 个人紧急医疗信息卡(过敏 / 血型 / 常用药)

## H. 文化 / 习惯 🌏
- [ ] 头巾 / 长袖长裤(中东 / 部分东南亚宗教场所)
- [ ] 拖鞋(进清真寺 / 印度庙 / 部分日式餐厅)
- [ ] 小礼物(商务拜访 / 寄宿家庭)
- [ ] 各国插座适配器(参见 C.)
- [ ] 文化禁忌预读(避免冒犯)
- [ ] 当地基础礼貌用语卡(纸质 / 截图)
- [ ] 当地给小费习惯(餐厅 / 酒店 / 司机)

## I. 其他 🎁
- [ ] 行李秤(防超重)
- [ ] 压缩袋(节省空间)
- [ ] 晾衣绳 / 衣夹
- [ ] 折叠水杯 / 保温杯
- [ ] 颈枕 + 眼罩 + 耳塞(飞机 / 大巴)
- [ ] 小零食(坚果 / 能量棒,防机场贵价)
- [ ] 笔 + 便签(填入境卡用)
- [ ] 充电宝 ≤ 100Wh(超规不能登机)
- [ ] 行李牌 + 备用钥匙
- [ ] 透明文件袋(放液体 / 充电宝过安检)
- [ ] 大量一元零钱(机场手推车 / 寄存柜)
- [ ] **孩子专属**(若同行):
  - 户口本 / 出生公证 / 委托书
  - 儿童常备药(退烧 / 止泻)
  - 零食 / 玩具 / 平板(长途)
  - 换洗衣物 × 2(液体泼洒)
  - 安全座椅(若租车)
