# 行程针对性问题模板(6 大场景)

> 用法:现场打开对应场景的清单,逐项打勾提问。提前 1-2 天邮件预约,效果最佳。

---

## 1. 学校探校 / 留学考察 🎓

### A. 报名 / 注册(签约日必问)
1. 当前最近入学点是几月?最晚报到日?
2. 学制是 1.5 年(AS+A2)还是 2 年?
3. 押金 / 注册 / 考试局(Cambridge / Edexcel)费 Year 1 total 多少钱?
4. 有没有 sibling / 早报 / 现金奖学金?(官网不写,问就完事)

### B. 学术 / 师资
5. 我的孩子意向科目组合(数 / 进阶数 / 物 / 经 / 中文)开班人数门槛?会不会不开?
6. 任课老师是否全职 Cambridge 培训师?外籍比例?
7. 模拟考 + 预测成绩 + UCAS 升学指导谁负责?从哪一年级开始?

### C. 生活 / 住宿
8. 住宿 / Homestay 提供吗?校方 vs 第三方?
9. 校服 / 教材 / 笔记本 购买渠道 + 二手市场?
10. 签证 + 监护人 一条龙谁负责?额外收费?

### D. 出口 / 背景
11. ECA / 体育 / 辩论 / MUN 强项?(影响大学 PS 含金量)
12. 最近 3 届 A-Level 毕业生去向统计(G5 / Russell / 本地公立)?

### 共同必问(目标本科大学)
- 录取底线等级是 A\*AA / AAA / AAB?
- Pre-U 直升协议要求什么 GPA / 等级?
- 奖学金截止日 + 门槛 GPA?
- A-Level → 本科最快几年?3+0 / 2+1?
- ACCA / CFA / BCS / 工程认证覆盖哪些专业?
- 带薪实习合作企业名单?

2. 会议地点(对方公司 / 第三方 / 线上 Zoom)?

5. 出发前 3 天再次确认航班 / 酒店?
6. 机场接送(对方派车 / 自己打车)?
7. 商务宴请的饮食禁忌?(清真 / 素食 / 过敏)
8. 名片 × 50 张 + 公司资料袋 + 礼品?

9. 当地手机卡 / 漫游流量包?
10. 当地货币 vs 美元 / 信用卡?宴请预算上限?

### D. 后续
12. 会议纪要 24h 内发出?
13. 跟进邮件 + 下次碰面时间?
14. 报销票据保存(发票 / 出租车 / 餐)?

## 6. 邮轮 🛳️

### A. 行前
1. 护照有效期 ≥ 6 个月(部分线路 ≥ 9 个月)?
2. 多次入境签证(经停港)?
3. 健康申报(邮轮公司提前 7 天交)?

### B. 上船
4. 登船港口 + 提前到达时间(通常 3h)?
5. 行李挂牌(大件托运 + 手提随身)?
6. 信用卡绑定船上账户?

### C. 船上
7. 自费项目(酒水 / 岸上游 / SPA)预算?
8. 自由用餐时间 vs 固定座位?
9. 儿童俱乐部 / 成人专区?

### D. 靠港
10. 岸上自由行 vs 跟团?
11. 当地 SIM / 货币 / 应急?
12. 回船时间(通常提前 1h)?

## 🎯 通用原则(适用所有场景)

1. **提前预约** — 大多数机构 / 餐厅 / 体验,提前 1-2 周 = 0 排队 / 价格更低
2. **带小礼物** — 当地特色小点心 / 茶 / 文创,初次见面很加分
3. **拍照存档** — 所有问到的信息拍照或笔记,避免重复
4. **诚实说"不知道"** — 现场问真问题,比"百度答案"更准
5. **结束时确认** — "今天我们聊过的 5 个要点:1...2...3...4...5...对吗?"
6. **24h 内发感谢信** — 邮件 / WhatsApp,简短真诚

## 📱 现场使用技巧

- 全部问题打印成 A4 1 张(16 号字)→ 现场翻阅不尴尬
- iPad / 手机装 PDF → 可现场打勾
- 录音(经对方同意)→ 回家整理纪要
- 带小本子 → 比手机显得更尊重
