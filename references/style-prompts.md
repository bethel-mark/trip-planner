# 风格 prompt 模板(Glassmorphism + Isometric + Soft 3D)

> 用途:丢给 GPT-4o / Midjourney / Doubao / 即梦 / MiniMax image-01 等 AI 出图。
> 英文 prompt 对所有模型表现最稳。

---

## 🟦 通用风格 prompt(可作前缀复用)

```
Editorial infographic in soft 3D + glassmorphism + isometric style,
macaron pastel color palette (lavender, mint, peach, cream, light blue),
matte pastel material rendering, no harsh shadows, soft top ambient lighting,
Apple Vision Pro spatial UI aesthetic, generous whitespace, clean modern
typography, render like a high-end Xiaohongshu lifestyle infographic.
1080x1080 square / 1080x1440 portrait / 1920x1080 landscape.
```

## 🟦 通用"行程图"模板(可改目的地复用)

```
A vertical long-screenshot travel infographic in 1080x2900px (mobile friendly),
Glassmorphism + Isometric + Soft 3D UI style.

Cover card at top with title: [TRIP_TITLE] in big bold Chinese.
Subtitle: [DATES] · 仅背包 · [TAGLINE].

Day sections separated by colored glass cards (Day 1 = teal, Day 2 = coral,
Day 3 = lavender, etc.). Each Day card has a large number badge on the
left.

Timeline events below each Day: vertical glass card per event with:
- Time badge (colored circle)
- Place name with emoji icon
- One-line description
- ⚠️ 注意 section with 3-6 bullet points (warm yellow accent)

Right side of each Day: isometric 3D scene with 2-4 emoji icons
(restaurant / hotel / transport / landmark) on soft 3D platforms.

Color palette: dark navy/purple gradient background with soft glow blobs,
glass cards white with low alpha, vibrant accent colors per Day.
```

## 🎯 调参技巧

| 想达到的效果 | prompt 关键词 |
|---|---|
| 更柔软 | soft, matte, pastel, no harsh shadow, ambient light |
| 更具科技感 | Apple Vision Pro, spatial UI, holographic |
| 更可爱 | rounded, chubby, kawaii, friendly, warm |
| 更专业 | minimalist, editorial, generous whitespace, clean type |
| 更小红书 | Xiaohongshu, lifestyle, editorial, soft pastel, infographic |
| 更杂志感 | magazine layout, asymmetric, large negative space |
| 配色强调 X 蓝 | dominant blue #A8D8EA, accent mint, secondary peach |
| 配色强调 X 暖 | dominant peach #F8D5C4, accent lavender, secondary cream |
